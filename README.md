## ✨ Features

- ✅ **Instant Analysis** - Enter any URL and get results in seconds
- ✅ **SEO Score** - Overall score from 0-100
- ✅ **Meta Tags Check** - Title and description optimization
- ✅ **Heading Structure** - H1-H6 analysis
- ✅ **Image Optimization** - Alt text validation
- ✅ **Link Analysis** - Internal vs external links
- ✅ **Actionable Recommendations** - Specific fixes to improve SEO

## 🛠️ How It Works

1. Enter a complete URL (including https://)
2. Click "Analyze URL"
3. Get instant SEO insights and recommendations

## 📊 What Gets Analyzed

| Feature | What We Check |
|---------|---------------|
| Title | Length (30-60 chars optimal) |
| Meta Description | Length (50-160 chars optimal) |
| Headings | Presence of H1, H2-H6 structure |
| Images | Alt text coverage |
| Links | Internal vs external ratio |
